About Us




This fun run is a friendly race for all people in Balingasag, Misamis Oriental and it is open for all ages. We are raising this fun run to donate the funds to some charity homes. Running in this fun run can also help you physically, there are 2 categories which are the 5 kilometer run and the second is the 8 kilometer run. We will also be giving out medals for those who are 1st to 5th and all the finishers. Encouraging you all to participate in this fun run because participating in this fun run would be such a big help to the charity homes we are giving the funds to, hopefully tons of people will join.
